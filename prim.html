Algoritmo de Prim

Encuentra el árbol de expansión mínima en un grafo conectado.

  • Atras

Instrucciones

Por favor ingresa la matriz de adyacencia del grafo. Cada valor representa el peso de la arista entre dos nodos. Usa 'Infinity' para representar la ausencia de arista.

Ingreso de datos

Introduce las filas de la matriz de adyacencia. Cada fila debe estar separada por comas y las filas deben estar separadas por saltos de línea. Ejemplo:

0, 2, 0, 6, 0
2, 0, 3, 8, 5
0, 3, 0, 0, 7
6, 8, 0, 0, 9
0, 5, 7, 9, 0
